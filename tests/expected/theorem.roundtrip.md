# Theorems

::: {#thm-max .theorem title="Maximum Principle"}
Let $u$ be harmonic on $\Omega$. Then $u$ attains its maximum on the boundary $\partial\Omega$.
:::

::: {#lem-bound .lemma}
For all $x \in [0,1]$, we have $f(x) \leq M$.
:::

::: {#def-cont .definition title="Continuity"}
A function $f$ is continuous at $a$ if $\lim_{x \to a} f(x) = f(a)$.
:::

::: {#cor-max .corollary}
If $u$ is harmonic and bounded, then $u$ is constant.
:::

::: {#prop-unique .proposition}
The solution is unique.
:::

::: {#conj-prime .conjecture}
There are infinitely many twin primes.
:::

::: {#rem-note .remark}
This result generalizes to higher dimensions.
:::

::: {#ex-simple .example}
Consider $f(x) = x^2$ on $[0,1]$.
:::
